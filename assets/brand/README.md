# HuddleUp Brand Kit

All visual identity for the HuddleUp project lives here. SVGs are the source of truth — raster exports are derived.

## Files

```
assets/brand/
│  Vector sources (edit these)
├── logo.svg                      # Primary master — indigo on transparent
├── logo-mono-dark.svg            # Single-colour ink for greyscale
├── logo-mono-white.svg           # White for dark backgrounds
├── logo-currentcolor.svg         # Inherits currentColor (theme-aware)
├── favicon.svg                   # 32-unit, optimised for tiny sizes
├── logo-horizontal.svg           # Icon + wordmark, light bg
├── logo-horizontal-dark.svg      # Icon + wordmark, dark bg
├── og-image.svg                  # 1200x630 social preview
├── github-social.svg             # 1280x640 GitHub social preview
├── vscode-marketplace-icon.svg   # 128x128 padded square
│
│  Generated rasters (don't edit — run `npm run build:assets` to regenerate)
├── favicon.ico                   # 16/32/48 multi-resolution
├── apple-touch-icon.png          # 180x180
├── og-image.png                  # 1200x630
├── github-social.png             # 1280x640
│
│  Docs
├── brand.md                      # Full brand guidelines (colours, type, do/don'ts)
└── README.md                     # This file
```

Plus the VS Code Marketplace icon, which lives next to the extension manifest:
```
vscode-huddleup/media/marketplace-icon.png   # 128x128 (regenerated by the same script)
```

## Quick links

- **Brand guidelines:** [`brand.md`](./brand.md)
- **Primary colour:** Indigo `#6366F1`
- **Wordmark rule:** `Huddle` in ink + `Up` in indigo, always camel-case as **HuddleUp**

## How to use in code

### In Markdown (e.g. main README)

```markdown
<p align="center">
  <img src="assets/brand/logo.svg" width="120" alt="HuddleUp">
</p>
```

### In HTML

```html
<img src="/assets/brand/logo.svg" alt="HuddleUp" width="48" height="48">
```

### In a VS Code extension (theme-aware activity bar icon)

Use `logo-currentcolor.svg` — VS Code recolours it to match the active theme.

## Exporting to PNG / ICO

See [`brand.md`](./brand.md#exporting-raster-assets) for `rsvg-convert` and ImageMagick commands.
